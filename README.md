# やること（ToDoアプリ）

iPhone風のシンプルなToDoアプリです。

- タスクの追加・削除・完了チェック
- ブラウザを閉じてもデータが残る（localStorage）
- スマホでも使える

## 使い方

GitHub Pages で公開すると、次のようなURLで開けます。

**https://【あなたのユーザー名】.github.io/【リポジトリ名】/**

例：リポジトリ名を `todo-app` にした場合  
→ `https://ouble.github.io/todo-app/`

詳細はフォルダ内の **「GitHubで公開する手順.txt」** を参照してください。
